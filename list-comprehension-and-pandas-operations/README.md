# List Comprehension & Pandas Operations

## About
This case study covers list comprehension structures and pandas operations using real-world datasets (car_crashes, Titanic, Tips) from the Seaborn library.

## Topics Covered
- List comprehension with conditional logic
- Column name manipulation (uppercase, prefix, suffix)
- Filtering and selecting columns
- Pandas data exploration (shape, nunique, dtype)
- Data type conversion
- Filtering with multiple conditions
- Handling missing values (fillna, mode, median)
- GroupBy and aggregation operations
- Apply and lambda functions
- Creating new columns

## Datasets
| Dataset | Source | Description |
|---|---|---|
| car_crashes | seaborn | US car crash statistics by state |
| titanic | seaborn | Titanic passenger survival data |
| tips | seaborn | Restaurant tipping data |
