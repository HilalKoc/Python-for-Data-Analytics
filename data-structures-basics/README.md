# Python Data Structures Basics

## About
This case study covers fundamental Python data structures and operations through 8 hands-on tasks.

## Topics Covered
- Data types (int, float, complex, str, bool, list, dict, tuple, set)
- String manipulation (upper, replace, split)
- List operations (indexing, slicing, pop, append, insert)
- Dictionary operations (keys, values, update, pop)
- Functions (return, even/odd separation)
- Enumerate usage
- Zip usage
- Set operations (issuperset, intersection, difference)

## Tasks
| Task | Description |
|------|-------------|
| Task 1 | Examining data types of given values |
| Task 2 | String manipulation operations |
| Task 3 | List operations |
| Task 4 | Dictionary operations |
| Task 5 | Function that separates even and odd numbers |
| Task 6 | Printing student rankings by faculty using enumerate |
| Task 7 | Printing course information using zip |
| Task 8 | Set operations with a custom function |

## Tools & Libraries
- Python 3.x
- No external libraries required
